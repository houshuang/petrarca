# This Week in AI Research

## Claude Code Goes Mainstream

Anthropic's Claude Code has seen explosive adoption among professional developers. The CLI-based coding assistant now supports autonomous multi-file editing, background agents, and a hooks system for customizing behavior.

Key developments this week:

- **Background agents**: Claude Code can now spawn long-running tasks that continue while you work on other things
- **MCP integration**: Model Context Protocol allows Claude Code to connect to external tools and data sources
- **Hooks system**: Custom shell commands triggered by specific events (file saves, tool calls, etc.)

## Open Source Model Updates

Several notable open source releases:

- **Qwen 3.5**: Alibaba released a new mixture-of-experts model competitive with GPT-4 on most benchmarks
- **Llama 4 Scout**: Meta's latest model shows strong performance in agentic tasks
- **DeepSeek R2**: Impressive reasoning capabilities at lower parameter counts

## Research Highlights

A paper from Stanford introduces "Thought Tokens" — a method for training models to use explicit reasoning tokens during inference without chain-of-thought prompting. Early results show 15-20% improvement on math benchmarks.

Another notable paper: "Scaling Test-Time Compute" from Google DeepMind demonstrates that giving models more time to "think" during inference can be more efficient than scaling model parameters.
